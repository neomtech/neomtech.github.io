 1. 패키지 구하기

다음의 장소에서 패키지를 다운로드 합니다.

 - http://ezne.net

 2. 패키지 압축 해제

먼저, 패키지를 시스템의 적절한 위치에 옮겨 놓도록 합니다.
패키지는 경로를 포함하여, 긴 파일이름으로 압축되어있기 때문에 윈집과
같은 프로그램으로 압축을 풀어야 합니다.

정상적으로 압축을 해제했다면 다음과 같은 구성을 볼 수 있습니다.


사용자 계정
     |
     +-- ez2000 (777 / 일부 계정에서는 755)
            |
            +-- system (777, 파일들은 모두 666)
                   |
                   +-- image
                   +-- help
                   +-- skins (스킨 디렉토리)
                   |     |
                   |     +-- note
                   |     +-- memo
                   |     +-- cacoG1
                   |
                   +-- db (777, 이지보드실행시 자동생성)
                   |
                   +-- users(777, 파일들은 모두 666)
                         |
                        +-- group (777, 파일들은 모두 666)




 3. 설치

① 위에서 구성된 디렉토리를 통째로 사용자 계정에 업로드합니다.
계정에 올릴때 반드시 ascii모드가 아닌 binary모드로 전송해야합니다.
                     ~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

② 다음과 같이 퍼미션을 설정해줍니다.
. ezadmin.cgi, ezboard.cgi, ezman.cgi의 퍼미션을 755로 설정
. system 디렉토리 퍼미션은 777로 설정
. system/users 디렉토리 퍼미션 777로 설정
. system/users 내의 모든 파일들을 666으로 설정
. system/users/group 디렉토리 퍼미션을 777로 설정
. system/users/group 내의 모든 파일들을 666으로 설정

* 퍼미션을 설정하는 방법이 어려우신 분들은 http://ezne.net의 cgi자료실
  에서 이지ftp를 받은후 다음과 같이 퍼미션을 변경해주면 됩니다.

  . 이지ftp에서 계정에 접속후 ez2000로 이동후 마우스 오른쪽 버튼을
    눌러 '이지보드 퍼미션' 메뉴 실행
  . system 디렉토리에 커서를 위치한후 'DB퍼미션설정'버튼을 누른다.
  . ftp에서 퍼미션 변경이 가능한 계정이라면 위 방법으로 퍼미션이 설정
    됩니다.

───────────────────────────────────
* 퍼미션 설정 방법

- 텔넷 : 일단 이지보드가 설치된 디렉토리에 들어갑니다.
    cd ez2000
    chmod 755 ez*.cgi
    chmod 777 system
    cd system


- ftp : 사용하고 있는 ftp프로그램에 따라 다르겠지만, 일단 파일 속성변
경 메뉴가 있는지 찾아서 다음과 같이 설정합니다.

    Owner - Read[X]   Write[X]   Execute[X]
    Group - Read[X]   Write[ ]   Execute[X]
    Other - Read[X]   Write[ ]   Execute[X]
───────────────────────────────────

③ system.cgi 수정
ez2000 디렉토리에 들어있는 system.cgi를 적당한 편집기로 열어서 다음 부
분을 수정해줍니다.

  * InitPass=0000

InitPass : 새로운 게시판이 생성될때 설정되는 초기 비밀번호

굳이 수정할 필요는 없지만, 관리자가 아닌 사람도 0000을 입력하여 관리자
메뉴에 들어갈수 있습니다.
이곳을 수정하셔도 되고, 관리자 메뉴에 들어가서 해당 게시판의 비밀번호를
바꿔도 됩니다.


* 주의 사항

일부 계정에서는 CGIWRAP을 지원하는 곳이 있습니다.
이런 경우에는 아이콘및 도움말이 보이지않고, 글작성이 되지 않는 문제가
발생합니다.

이지보드를 다음과 같이 ez2000디렉토리와 system디렉토리를 나누어 설치하
기 바랍니다.

───────────────────────────────────
사용자 계정
     |
     +-- cgi-bin -- ez2000 (ez2000 디렉토리내의 파일을 모두 복사)
     |
     +-- ez2000 (777 / 일부 계정에서는 755)
            |
            +-- system (777, 파일들은 모두 666)
                   |
                   +-- image
                   +-- help
                   +-- skins (스킨 디렉토리)
                   |     |
                   |     +-- note
                   |     +-- memo
                   |     +-- cacoG1
                   |
                   +-- db (777, 이지보드실행시 자동생성)
                   |
                   +-- users(777, 파일들은 모두 666)
                         |
                        +-- group (777, 파일들은 모두 666)
───────────────────────────────────

퍼미션 설정은 위에서 언급한대로 해주면 됩니다.
그런후, ez2000 디렉토리에 들어있는 system.cgi를 적당한 편집기로 열어서
다음과 같이 수정해줍니다.
예 1) SystemPath=/user/cherie/public_html/ez2000/system
예 2) SystemPath=../../system
등과 같이 절대 경로를 적어줍니다.

예 1) CGIWRAPSystemPath=http://사용자 홈페이지/ez2000/system
이 경로는 다운경로및 이지보드의 아이콘이 제대로 출력되도록 하기위한 경
로설정 항목입니다.

* CGIWRAP계정이 아닌 경우에는 절대로 SystemPath / CGIWRAPSystemPath를
수정하지 마십시오(대부분 계정들은 CGIWRAP계정이 아닙니다.).

───────────────────────────────────
. 사용 방법
───────────────────────────────────

 1. 실행 방법

http://사용자 홈페이지/ez2000/ezboard.cgi?db=db명

위에서 db명이라 함은, 게시판 데이타를 저장할 디렉토리를 설정하는 것으
로, 되도록 영문과 숫자만을 사용해서 이름을 설정해주는 것이 바람직합니다.
한글명으로 지정하는 경우, 일부 서버에서 에러가 생길 수도 있습니다.

예) DB명 설정 예제
http://사용자 홈페이지/ez2000/ezboard.cgi?db=free
http://사용자 홈페이지/ez2000/ezboard.cgi?db=guest

 2. 게시판 추가

. http://사용자 홈페이지/ez2000/ezman.cgi 를 실행합니다.
. 처음 실행시 신규 비밀번호 등록화면이 뜨는데, DB매니저의 비밀번호는 루
트 비밀번호로서 모든 게시판의 관리자 메뉴 접속, 글수정/삭제등이 기능하므
로 다른 사람이 알수 없도록 주의해서 비밀번호를 등록하기 바랍니다.
. 정상적으로 비밀번호가 등록되면 다시 비밀번호를 묻는 화면이 뜨게되는데,
이때 등록한 비밀번호를 입력합니다.
. 적당한 DB명을 입력한후 생성버튼을 누르면 게시판이 추가됩니다.

 3. 참고사항

* 홈페이지에 삽입하려면...
사용하고 있는 HTML문서내에 다음과 같이 포함해주면 등록된 게시판을 사용할
수 있습니다.
<A HREF='http://사용자 홈페이지/ez2000/ezboard.cgi?db=db명'>게시판</a>

* DB매니저 비밀번호를 잊어버렸다면...
DB매니저 비밀번호는 system 디렉토리내에 password.cgi 파일로 저장되어 있
습니다. password.cgi를 지우고, DB메니저를 실행하면 다시 비밀번호를 등록
하는 메뉴가 나타납니다.

* 관리자 비밀번호를 잊어버렸다면...
관리자 비밀번호는 db명의 디렉토리내에 password.cgi 파일로 저장되어 있습
니다. password.cgi를 지우고, 관리자 메뉴로 접속하면 system.cgi에 설정된
기본 암호로 초기화됩니다.
기본암호를 사용하여 관리자 메뉴에 접속한후 새로운 비밀번호로 바꿀 수 있
습니다.

* 메일서버 주소 설정은...
SendMail을 지원하는 계정이라면 기본값인 127.0.0.1을 설정하면 됩니다.
만일 127.0.0.1로 메일 발송이 안되는 경우(윈도우즈버전)에는 공개된 메일
서버 주소를 적어주면 됩니다.

* 회원관리 기능을 사용하려면
. /ez2000/ezmem.cgi 퍼미션을 755로 설정합니다.
